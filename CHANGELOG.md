# Changelog de SAVE FILE

## 1.0.0

Primera versión pública del proyecto.

### Editor

- Bloc de notas local con título, cuerpo y estado de ánimo.
- Creación, edición y borrado de entradas.
- Conteo de palabras en tiempo real.
- Animación del último carácter escrito.
- Resaltado cromático de palabras emocionales mediante una transformación visual local.
- Medidor de combo de escritura.
- Mensaje animado al guardar.

### Audio y respuesta táctil

- Sonido de tecla suave.
- Sonido de cierre de oración.
- Sonidos de guardado, misión y subida de nivel.
- Vibración breve configurable.
- Controles independientes para sonido y vibración.

### Sistema de juego

- Experiencia y niveles.
- Tres misiones diarias.
- Recompensas de XP.
- Habitación pixel art evolutiva.
- Cuatro paletas desbloqueables.
- Entradas presentadas como slots de guardado.

### Datos y privacidad

- Base de datos Room local.
- Preferencias locales.
- Sin cuentas, anuncios, nube o analítica.
- Copia de seguridad y transferencia mediante Android.

### Construcción

- Workflow de GitHub Actions para pruebas y APK debug.
- Workflow manual para publicar GitHub Releases.
- Firma debug fija para actualizaciones de desarrollo.
