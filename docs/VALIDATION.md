# Validación

La estructura del proyecto fue revisada para comprobar:

- rutas de paquetes y recursos;
- XML del manifiesto y reglas de respaldo;
- YAML de GitHub Actions;
- archivos de audio WAV;
- versión y applicationId;
- presencia del wrapper de Gradle;
- pruebas unitarias para niveles y misiones.

La compilación completa debe ejecutarse en GitHub Actions o en un equipo con acceso a los repositorios de Gradle y Android. El entorno donde se generó este paquete no tiene acceso de red para descargar la distribución de Gradle.
