# SAVE FILE 1.0.0

La primera versión convierte un bloc de notas en una pequeña partida personal. Las entradas son slots, las consignas son misiones y el progreso transforma una habitación pixel art.

## Incluye

- Editor local de notas.
- Misiones diarias.
- XP y niveles.
- Animaciones y sonidos de escritura.
- Estados de ánimo.
- Habitación evolutiva.
- Cuatro temas.
- GitHub Actions para generar el APK.

## Instalación

Descarga `SaveFile.apk`, habilita la instalación de aplicaciones desde el navegador o gestor de archivos utilizado y abre el APK.
